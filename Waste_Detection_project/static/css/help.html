How To Predict Waste and Aquatic Objects

Follow these steps to use the website effectively and get accurate predictions.

1. Upload Image

Use Select Image or drag-and-drop an image into the upload area. Supported formats are JPG, PNG, and WEBP.

2. Wait for Processing

The scan progress bar indicates that visual analysis is running. Results appear automatically after processing.

3. Review Prediction

The uploaded image is shown on the left. The predicted image with bounding boxes appears on the right after analysis.

4. Filter Results

Use Quick Filter in Analysis Result to switch between Waste and Aquatic detections. Scroll the container to view all matched items.

5. Download or Reset

Use Download Result to export the predicted image with boxes, or Clear / Upload New Image to start another prediction.
